# 🚀 공고뚝딱 (Research Banner Builder)
> **"연구원과 마케터를 위한 1분 완성 지능형 공고 배너 생성기"**

실무에서 반복되는 공고 배너 제작 업무의 비효율을 해결하기 위해 기획된 웹 서비스입니다. 디자인 규격을 준수하면서도 누구나 고퀄리티의 배너를 쉽고 빠르게 제작할 수 있도록 돕습니다.

## 🛠 주요 기능 (Key Features)

### 1. 지능형 레이아웃 엔진 (Smart Layout Engine)
- **규격 강제 최적화:** 인스타그램 피드(1080x1350px) 및 스토리(1080x1920px) 규격에 맞춘 자동 렌더링.
- **Auto-Shrink 로직:** 텍스트 양이 많아지면 [제목 축소 → 간격 조정 → 본문 축소] 순으로 레이아웃을 실시간으로 최적화합니다.
- **실시간 경고 시스템:** 규격을 벗어날 경우 상단 알림 띠를 통해 즉각적인 피드백을 제공합니다.

### 2. 마케터 친화적 UX (Marketer-friendly UX)
- **항목 프리셋:** 참여 대상, 진행 일시, 신청 마감 등 실무에서 반드시 포함되는 6가지 항목을 드롭다운으로 간편하게 선택합니다.
- **리워드 이미지 매핑:** 네이버페이, 커리어톡 등 경품 선택 시 전용 일러스트가 배너에 자동으로 배치됩니다.
- **스마트 파일명:** 다운로드 시 날짜와 규격이 포함된 파일명(`YYMMDD_공고뚝딱_feed.png`)을 자동으로 생성하여 파일 관리의 효율을 높였습니다.

### 3. 편집 및 상태 관리
- **드래그 앤 드롭:** 항목 간의 순서를 자유롭게 재배치할 수 있는 인터랙션.
- **임시 저장(LocalStorage):** 브라우저를 닫아도 마지막 작업 내역이 유지되어 작업의 연속성을 보장합니다.
- **편집 모드 토글:** 수정 시에만 편집 도구가 나타나도록 설계하여 작업 중 실수를 방지합니다.

## 🎨 Design Variants
- **Template A (White):** 깔끔하고 신뢰감 있는 기본 디자인.
- **Template B (Signature Red):** 강렬한 테마 컬러와 함께 배경색에 맞춰 로고가 화이트로 자동 전환되는 스마트 브랜딩 적용.

## 💻 Tech Stack
- **Frontend:** HTML5, CSS3, JavaScript (Vanilla JS)
- **Library:** Sortable.js (Drag & Drop)
- **Deployment:** Vercel
